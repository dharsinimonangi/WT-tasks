# Web Technologies Lab Tasks

This repository contains the tasks completed as part of the **Web Technologies Course**.

## Tasks

### Task 1 - Portfolio Website

A personal portfolio website developed using HTML and CSS.

#### Features

* Responsive Design
* Home Section
* About Me Section
* Skills Section
* Projects Section
* Contact Information
* Modern User Interface

---

### Task 2 - Student Profile Web Page

A responsive Student Profile webpage developed using HTML5 and Bootstrap 5.

#### Features

* Bootstrap Navbar
* Student Profile Card
* Academic Performance Table
* About Section
* Skills Section
* Contact Section
* Responsive Design using Bootstrap Grid
* Footer

---

### Task 3 - Personal Portfolio Website with Contact Form Validation

A responsive personal portfolio website developed using HTML, CSS, and JavaScript.

#### Features

* Navigation Bar
* Home Section
* About Me Section
* Skills Section
* Education Section
* Projects Section
* Contact Section
* Profile Image
* Responsive Layout using Flexbox/Grid
* Footer with Social Media Links

#### Contact Form Validation

* Name cannot be empty
* Email validation
* Subject must contain at least 5 characters
* Message must contain at least 20 characters
* Error messages displayed beside fields
* Success alert after valid submission
