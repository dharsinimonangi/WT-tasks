### Task 3 - Personal Portfolio Website with Contact Form Validation
A responsive personal portfolio website developed using HTML, CSS, and JavaScript.

#### Features
- Navigation Bar
- Home Section
- About Me Section
- Skills Section
- Education Section
- Projects Section
- Contact Section
- Profile Image
- Responsive Layout using Flexbox/Grid
- Footer with Social Media Links

#### Contact Form Validation
- Name cannot be empty
- Email validation
- Subject must contain at least 5 characters
- Message must contain at least 20 characters
- Error messages displayed beside fields
- Success alert after valid submission

---

### Task 4 - Student Registration Form with Validation
A responsive student registration form developed using HTML, CSS, and JavaScript.

#### Features
- Full Name Field
- Roll Number Field
- Email ID Field
- Mobile Number Field
- Password & Confirm Password Fields
- Gender Selection
- Department Selection
- Date of Birth Selection
- Responsive User Interface

#### Validation Rules
- Full Name should not be empty
- Roll Number must contain only letters and numbers
- Email must be in a valid format
- Mobile Number must contain exactly 10 digits
- Password must be at least 8 characters long
- Password and Confirm Password must match
- Gender must be selected
- Department must be selected
- Date of Birth should not be empty
- Error messages displayed for invalid inputs
- Success message displayed after successful validation
